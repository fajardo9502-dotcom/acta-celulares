# Sistema Web para Actas de Entrega de Celulares

Sistema web desarrollado durante mi práctica empresarial en **Colombiana de Comercio S.A. (Corbeta / Alkosto)** para automatizar y digitalizar el proceso de asignación y entrega de equipos móviles a los colaboradores de la compañía[cite: 1].

##  Características del Proyecto
- **Formulario Web Dinámico:** Captura de datos del empleado y del dispositivo (IMEI, modelo, línea corporativa).
- **Módulo de Firma Digital:** Permite al trabajador firmar directamente en la interfaz táctil o con el mouse.
- **Persistencia de Datos:** Conexión y almacenamiento automatizado en hojas de cálculo (Excel) mediante la librería `openpyxl`.
- **Generación de Reportes:** Creación automática de documentos PDF oficiales con el formato institucional e integración de la firma capturada.
- **Control de Versiones:** Ciclo de desarrollo gestionado a través de Git y GitHub bajo buenas prácticas.

##  Tecnologías Utilizadas
- **Backend:** Python, FastAPI
- **Frontend:** HTML5, CSS3 (Estructura adaptada a la identidad corporativa), JavaScript
- **Librerías Clave:** `openpyxl` (Manejo de bases de datos en Excel), herramientas de renderizado PDF.

##  Impacto del Proyecto
El sistema reemplaza un flujo de trabajo 100% manual, reduciendo los tiempos de procesamiento en la generación del acta física, centralizando la información en un formato digital estructurado y eliminando los errores comunes de digitación manual.
